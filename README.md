# RAG-Powered Website Chatbot

A chatbot that can ingest any website URL, recursively scrape relevant pages, and answer user questions using Retrieval-Augmented Generation (RAG).

## Features

- Recursive website crawling
- Text extraction and cleaning
- Semantic search
- Grounded answer generation
- Source citations

## Tech Stack

- Python
- BeautifulSoup
- LangChain
- Sentence Transformers
- ChromaDB
- Groq
- FastAPI
- Streamlit
